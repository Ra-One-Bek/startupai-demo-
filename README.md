# startupai
1. Суть продукта (Elevator Pitch)
Веб-сервис, который анализирует рынок вакансий (HH.kz) и резюме пользователя, находит разрыв в навыках (Gap) и строит персональный план обучения, чтобы пользователь мог претендовать на зарплату выше рынка.

2. Целевая аудитория
Junior/Middle специалисты (начали с Data Analysts).

Люди, которые хотят повышения зп, но не знают, что именно учить.

3. Функционал MVP (Что делаем в первой версии)
Только самое необходимое, чтобы сервис заработал.

А. Сбор данных (Backend):

[ ] Ежедневный парсинг вакансий "Data Analyst" с HH.kz (пока без Enbek).

[ ] Очистка данных и извлечение навыков через GPT (Skill Extraction).

[ ] Сохранение в базу Postgres.

Б. Пользователь (Frontend/Bot):

[ ] Загрузка резюме в PDF.

[ ] Выбор целевого грейда (Junior/Middle).

[ ] Кнопка "Построить трек".

В. Результат (Output):

[ ] Список: "Твои навыки" vs "Требования рынка".

[ ] Текст от ИИ: План обучения по неделям (текстом).

[ ] (Опционально) Ссылки на бесплатные курсы (YouTube).

4. Чего НЕ БУДЕТ в MVP (Anti-Scope)
Чтобы не распыляться, мы договариваемся ЭТОГО пока не делать:

❌ Мобильного приложения (будет только Telegram-бот или простой сайт).

❌ Платная подписка и платежная система.

❌ Личный кабинет с сохранением прогресса (пока одноразовый анализ).

❌ Интеграция с Coursera/Udemy.

❌ Другие профессии (дизайнеры, маркетологи — потом).

5. Стек технологий
Язык: Python 3.12

API: FastAPI

ETL: Airflow (или Schedule script на старте)

AI: OpenAI GPT-4o-mini

DB: PostgreSQL + pgvector

Interface: Telegram Bot (aiogram)

6. Критерий успеха (Definition of Done)
MVP считается готовым, когда:

Я могу скинуть ссылку другу.

Он кидает свой PDF.

Бот в ответ присылает адекватный план обучения, основанный на реальных вакансиях с HH.kz.
