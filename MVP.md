# 🚀 ТЗ: Career AI Navigator (MVP)

**Репозиторий:** [github.com/DarynRz/startupai](https://github.com/DarynRz/startupai)

### 1. Суть продукта (Value Proposition)
Платформа анализирует текущий рынок вакансий (HH.kz) и строит для пользователя индивидуальный образовательный трек.  
**Цель:** Показать пользователю кратчайший путь (какие именно навыки подтянуть), чтобы получить оффер с зарплатой выше рынка.

---

### 2. Целевая аудитория
* **Кто:** Junior/Middle Data Analysts (в будущем другие IT-сферы).
* **Проблема:** "Я знаю Python, но меня не берут на работу / мало платят. Что мне учить: Airflow, Tableau или A/B тесты?"

---

### 3. Функционал MVP (Scope)

#### А. Сбор данных (Backend & ETL)
* [ ] **Парсер HH.kz:** Скрипт ежедневно собирает вакансии по тегу "Data Analyst" (регион: Казахстан).
* [ ] **AI Cleaning:** GPT-4o-mini очищает описания вакансий, выкидывает "воду" и извлекает чистые хард-скиллы (JSON).
* [ ] **База данных:** Сохранение структуры вакансий в PostgreSQL.

#### Б. Пользовательский путь (User Flow)
* [ ] **Загрузка резюме:** Пользователь отправляет PDF файл в Telegram-бота.
* [ ] **CV Parsing:** ИИ извлекает текущие навыки пользователя.
* [ ] **Gap Analysis (Ядро):** Алгоритм сравнивает "Навыки юзера" vs "Топ требований рынка". Вычисляет разницу (Gap).

#### В. Результат (Deliverable)
* [ ] **Отчет:** Текстовое сообщение или PDF с пошаговым планом обучения (напр. "Неделя 1: Изучить Window Functions в SQL").

---

### 4. Anti-Scope (Чего НЕ делаем в MVP)
* ❌ Мобильное приложение (делаем только Telegram-бота).
* ❌ Веб-интерфейс (Frontend).
* ❌ Личный кабинет с историей прогресса.
* ❌ Платная подписка / биллинг.
* ❌ Интеграция с образовательными платформами (курсы не продаем, только советуем темы).

---

### 5. Стек технологий
* **Language:** Python 3.12
* **Backend:** FastAPI
* **Database:** PostgreSQL + pgvector (для векторного поиска)
* **AI/ML:** OpenAI API (GPT-4o-mini), LangChain
* **ETL:** Schedule (на старте), позже Airflow
* **Interface:** Telegram Bot API (`aiogram`)

---

### 6. Критерий готовности (Definition of Done)
MVP считается сданным, когда пользователь может скинуть PDF-резюме боту и получить в ответ адекватный список навыков, которых ему не хватает для средней зарплаты по рынку.
